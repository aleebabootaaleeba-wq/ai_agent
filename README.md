# Time-Telling Agentic AI
This project demonstrates a simple Agentic AI using Python.
The agent can fetch real-time information using an external API.
## Features
- Answers time-related questions
- Uses real-world time API
- Can run in terminal
## Technologies
- Python
- Requests library
- World Time API
## How to Run
1. Install requirements
2. Run `python time_agent.py`
